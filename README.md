Installation

1)Clone the repo
```sh
git clone https://github.com/Benyoubilel/tp2-vuejs-jsondb.git
```
2)cd to repository

3)Install NPM packages
```sh
npm install
```

4)install vuelidate and axios
```sh
npm install @vuelidate/core @vuelidate/validators
```
```sh
npm install axios
```
5)Start vue
```sh
npm run dev
```
